# Raspberry Pi 3のセットアップ

  Raspberry Piのプログラムを動かすためには以下をインストールする必要がある。
  - gcc: C/C++コンパイラ
  - CMake: Makefileの自動生成ツール、これを使うことでC/C++のコンパイルが楽になる

  gccをインストール
  ```bash
  $ sudo apt-get install build-essential
  ```

  CMakeをインストール
  ```bash
  $ sudo apt-get install cmake
  ```
